## Acquisition
Download

`git clone https://github.com/wick3dr0se/pkm; cd pkm`


## Execution
`bash pkm` or `./pkm`

## Usage
Naivgate with arrow keys, vim bindings (h,j,k,l) or w,a,s,d

Press Q or Ctrl+C to quit
